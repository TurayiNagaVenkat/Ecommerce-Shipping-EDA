# Ecommerce-Shipping-EDA
Kaggle E-Commerce Shipping Data (11,000+ records)

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Shipment mode strongly affects delivery delays
- Heavy products are more likely to be delayed
- Customer ratings remain stable due to discounts and support
- Ship and Road shipments have the highest delays

- ecommerce_eda.ipynb → Full analysis
- ecommerce_data.csv → Dataset
